
## 检查信息查询by检查类型

请求方式: GET
请求参数:

* 检查类型
* 页码
* 显示条数
* 查询老师

### 返回结果：
```js
{
    type: 检查类型
    有无填写检查结果：boolean
    检查时间：
    创建时间：
    result检查结果：[
        {
            id: .....唯一标识
            type: boolean | number | checkout
            title: 检查问题 eg：学生迟到人数
            value：检查问题对应的结果
            img: 拍照回传的照片 （可选）
            remarks: 备注（可选）
        }
    ]
}
```
